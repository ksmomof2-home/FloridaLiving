  <div class="plant-facts">
    <h3>Quick Facts</h3>
    <ul>
      <li><strong>USDA zones:</strong> 9–11 (protect below 28 °F)</li>
      <li><strong>Sun:</strong> Full sun</li>
      <li><strong>Water:</strong> Average — never bone-dry</li>
      <li><strong>Height in pot:</strong> 6–10 ft</li>
      <li><strong>Fruit:</strong> Year-round in 9b</li>
    </ul>
    <p><strong>Primary Use:</strong> Culinary → fresh lime juice, mojitos!</p>
  </div>
</div>

<div class="full-width">
  <h3>Wildlife Supported</h3>
  <ul>
    <li>Nectar for bees when flowering</li>
    <li>Occasional giant swallowtail host</li>
  </ul>

  <h3>Pests to Watch</h3>
  <ul>
    <li>Citrus leafminer (cosmetic only)</li>
    <li>Scale → wipe with alcohol</li>
  </ul>

  <h3>Unusual / Fun Facts</h3>
  <ul>
    <li>Flowers smell incredible</li>
    <li>Thornless varieties exist</li>
  </ul>

  <h3>Local Care Tips (The Villages 9b)</h3>
  <ul>
    <li>Fertilize every 6–8 weeks with citrus food</li>
    <li>Mulch with pine bark</li>
    <li>Wrap trunk + frost cloth below 30 °F</li>
    <li>Current status: Pos 4 — fertilized, foiled, mulched Nov 20</li>
  </ul>
</div>
